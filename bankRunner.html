How To Run

This code runs in the console. You can open that on Mac by pressing COMMAND + OPTION + K in Firefox, or COMMAND + OPTION + J in Chrome

You can run the following commands:
  • account = new Account - creates new account called 'account'
  • account.deposit(amount) - adds the specified amount to your account
  • account.withdraws(amount) - withdraws the specified amount from your account
  • account.printStatement() - lists all the transactions on your account
